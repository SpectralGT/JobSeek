# JobSeek

React + TypeScript job application form. Output matches the structure of `userData.json`.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`.

## Form output

- **Generate JSON** — Builds an object with the same keys and shape as `userData.json`.
- **Copy to clipboard** — Copies the JSON.
- **Download userData.json** — Downloads the file.
